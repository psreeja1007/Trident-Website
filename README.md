# Hostel 15 – Trident Website

This repository contains the code for the Hostel 15 Trident website, created as part of the Web GC organized by the Web Team, ITC, for the 2025–26 academic year.

## Getting Started

To run this project on your local machine, follow the steps below.

1. Clone the repository:
```bash
    git clone <repository-url>
```
2. Install the required packages(in the correct directory):
```bash
    npm install
```

4. Start the development server:
```bash
    npm start
```
This will launch the website on your local machine for development and testing.
